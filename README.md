Welcome to the **14 Days of Flask Challenge** repository! 🚀

This repository contains a structured, hands-on, day-by-day progression for learning **Flask** (Python Web Framework) from absolute fundamentals to building full-stack web applications with databases, authentication, and REST APIs.

Whether you are starting out with web development or using this as a quick revision guide, each daily folder is self-contained with working code, dependency lists, and documentation.

---

## 🧠 What You Will Learn

By exploring this 14-day challenge, you will gain hands-on experience with the following core concepts:

### 1. Core Web Fundamentals & Routing
- Dynamic URL routing (`/user/<username>`) and path parameters.
- Handling multiple HTTP verbs (`GET`, `POST`, `PUT`, `DELETE`).
- Managing request and response objects (`request.form`, `request.args`, `jsonify`, `render_template`).

### 2. Templating & Frontend Integration
- **Jinja2 Templating Engine:** Template inheritance (`{% extends %}`), control structures (loops, conditionals), and dynamic rendering.
- Serving static assets (CSS, JavaScript, Images) efficiently.

### 3. Database Management & ORM
- Relational database design using **Flask-SQLAlchemy** (ORM) with SQLite.
- Executing complete **CRUD** (Create, Read, Update, Delete) operations.
- Structuring database relationships (One-to-Many, Foreign Keys).

### 4. Authentication, Authorization & Security
- Secure password hashing and verification using `Werkzeug.security`.
- User session management and persistent logins with `Flask-Login`.
- Protecting sensitive data using environment variables (`python-dotenv`).
- Form validation and dynamic feedback using Flash messaging.

### 5. Production-Ready Software Architecture
- Modularizing codebases using **Flask Blueprints**.
- Implementing scalable application patterns like the **Application Factory** (`create_app()`).
